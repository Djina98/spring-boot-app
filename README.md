# Spring Boot App 
Spring Boot CRUD application for purposes of exam "Napredne softverske tehnologije".

## Techology stack
 - Spring MVC
 - Spring Data JPA
 - Hibernate
 - Powered by Spring Boot & Tomcat

## Database
 - HSQLDB (default)
 - MySQL

## Instructions
Run `mvn spring-boot:run` command. Application is available from http://localhost:8080.
